## Submission Strategy & Tips

### Prioritization Order
1. **Week 1-2**: Thailand-specific directories (#1-14) — highest relevance signals
2. **Week 3-4**: Blog aggregators & guest post pitches (#19-36) — traffic + authority
3. **Week 5-6**: General travel directories (#15-18) + platform profiles (#45-52)
4. **Week 7-8**: General web directories (#37-44) — fill in the long tail

### Best Practices
- **NAP Consistency**: Always use the same site name ("Go2Thailand.com" or "Go2 Thailand") and URL format across all submissions
- **Unique Descriptions**: Write a unique 2-3 sentence description for each directory — never copy-paste the same text
- **Category Selection**: Always choose the most specific category (e.g., "Asia > Thailand > Travel Guides" over just "Travel")
- **Avoid Spam Directories**: Skip any directory with DA < 20 or that looks abandoned/spammy
- **Track Submissions**: Use the checkboxes above to track progress
- **Monitor Backlinks**: Check Ahrefs/Moz monthly to verify which submissions resulted in live backlinks
- **Reciprocal Links**: Some directories require a backlink — only do this for high-quality, relevant sites

### Description Template
> Go2Thailand.com is a comprehensive Thailand travel guide featuring detailed city guides for 33 destinations, interactive budget and weather indexes, itineraries, local food guides, and practical travel tips. Updated daily with fresh content in English and Dutch.

### What NOT to Submit To
- Directories with DA < 15
- Sites that are clearly link farms (hundreds of outbound links per page)
- Directories that haven't been updated in 2+ years
- Any site asking for payment just to be "reviewed"
- Auto-approve directories with no editorial standards
