# Pinterest Attraction Batch Design

## Goal

Create a ready-to-use Pinterest content batch for the 19 newly added attraction pages, stored in a clearly named subfolder inside the existing `public/images/pinterest` directory.

## Output

- A new folder: `public/images/pinterest/2026-04-attraction-repair-batch/`
- A human-readable markdown file with one primary SEO-oriented pin brief per page
- A CSV export with the same data for faster copy/paste or spreadsheet use
- A short future-run candidate file for the next Pinterest batch

## Content Rules

- One primary pin angle per new page
- Titles optimized for search intent, readability, and clickability without keyword stuffing
- Descriptions written in English to match the target page URLs and broader Pinterest travel search demand
- Softer pages like accommodation, dining, and seasonal events should use editorial framing rather than hard selling
- Each pin includes a target URL, overlay suggestion, filename suggestion, image prompt, and keyword focus

## Scope Boundary

- No image files are generated in this task
- No automation changes to the Pinterest posting system
- No mass update of existing Pinterest assets outside the new batch folder
