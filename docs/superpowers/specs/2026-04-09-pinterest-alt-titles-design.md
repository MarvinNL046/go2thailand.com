# Pinterest Alt Titles Design

## Goal

Add a second Pinterest support file for the 19 attraction pages that already have primary pin copy, focused only on alternative title testing.

## Output

- One markdown file with 3 alternative SEO-friendly Pinterest titles per page
- One CSV file with the same title variants for quick spreadsheet or workflow use
- Both stored inside the existing batch folder: `public/images/pinterest/2026-04-attraction-repair-batch/`

## Rules

- Titles should stay readable and click-oriented
- Titles should remain Pinterest- and SEO-friendly without keyword stuffing
- Variants should give different angles where possible: destination-first, problem/guide angle, and visual/discovery angle
- Keep titles comfortably under Pinterest title limits
